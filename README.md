# **Desafio  Programa Gente que Soma**

Para resolver o desafio foi usado **Reactjs**


# Instalação

Para instalar execute no terminal dentro da pasta do projeto

    $npm install

ou 

    $yarn install

## Execução

A execução é feita com o comando

    $npm start
    
   ou
		
    $yarn start
também dentro da pasta do projeto
